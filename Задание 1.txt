Необходимо выбрать одну из тем и написать программу, получающую по API из открытых источников данные для решения соответствующей задачи ML. Необходимо выделить целевой признак для обучения с учителем. Критерии оценки:
1. Соответствует ли набор признаков в датасете исходной задаче?
2. Соответствует ли целевой признак исходной задаче?
Темы:
1. Построение модели для оценки кредитоспособности клиентов банка.
2. Анализ пользовательских оценок для предсказания популярности фильмов.
3. Прогнозирование стоимости квартир/домов по их характеристикам.
4. Классификация риска возникновения детских заболеваний на основе медицинских карт.
5. Прогноз температуры воздуха и других погодных показателей по регионам.
6. Анализ факторов, влияющих на владение автомобилем среди населения.
7. Выявление мошеннических финансовых транзакций с помощью машинного обучения.
8. Прогнозирование востребованности профессий по рынку труда.
9. Анализ текстовых отзывов для выявления удовлетворенности клиентов.
10. Прогнозирование объёмов продаж продуктов питания по категориям и сезонам.
11. Анализ ответов на социологические опросы для изучения общественных настроений.
12. Прогнозирование результатов спортивных матчей по статистике команд.
13. Анализ поведения пользователей в социальных сетях для выявления трендов.
14. Определение факторов, влияющих на оборот розничных магазинов.
15. Прогнозирование успеваемости учащихся по комплексу их характеристик.
16. Анализ и прогнозирование потребления электроэнергии в жилых и коммерческих зданиях.
17. Исследование демографических факторов, влияющих на различные сферы жизни.
18. Анализ и прогнозирование дорожных происшествий по историческим данным.
19. Связь между образом жизни, питанием и состоянием здоровья населения.
20. Анализ покупательских корзин и прогнозирование повторных покупок в интернет-магазинах.
21. Прогнозирование туристического потока в регионы и загрузки гостиниц.
22. Анализ зависимости уровня заработной платы от профессии и региона.
23. Прогноз популярности мобильных приложений по их метаданным и рейтингам.
24. Анализ обращений клиентов в колл-центры и оценка их удовлетворенности.
25. Построение диагностической модели выявления заболеваний по медицинским данным.
26. Выявление закономерностей использования интернета среди различных групп населения.
27. Прогнозирование уровня загрязнения воздуха в городах.
28. Прогноз урожайности сельскохозяйственных культур по климатическим и почвенным данным.
29. Анализ условий, влияющих на рейтинг университетов и карьеру выпускников.

30. Анализ эффективности рекламных кампаний на основе данных о затратах и откликах.
